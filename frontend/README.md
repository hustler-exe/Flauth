# Flauth Frontend 🎨 - React SPA

## ⚙️ Tech Stack
- React + Vite
- React Router
- Axios
- Vanilla CSS

## 🧩 Features
- Login & Signup UI
- Auth with JWT
- Protected routes
- API integration

## ⚡ Setup

# 1. Install dependencies
npm install

# 2. Start dev server
npm run dev

# 3. Environment variable
# Create a .env file:
VITE_API_URL=http://localhost:5000
